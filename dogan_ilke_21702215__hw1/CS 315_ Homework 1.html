Boolean Expressions in C, Go, Javascript, PHP, Python, and Rust

Expressions are the fundemental means of specifying computations in a programming language. Expressions involve operators and operand.

In this homework assignment, you will compare boolean expressions provided in six different programming languages: C, GO (Golang), Javascript, PHP, Python, and Rust. You will investigate how the following design issues are addressed in these programming languages:

  1. Boolean operators provided
  2. Data types for operands of boolean operators
  3. Operator precedence rules
  4. Operator associativity rules
  5. Operand evaluation order
  6. Short-circuit evaluation

For each design issue and for each language, explain the choices selected by these languages. Support your explanation by giving examples using code segments in these languages. You can illustrate the answers to these questions, in different parts of a single program. The example program must be complete, so that they can be executed directly. The files must execute without error or warning.

You can use online compiler/interpreters for this homework. For each language, prepare a single source code file that examplifies and tests each issue, in the order given above. Your example programs must be different that the code examples that may be available in the Internet.

Organize all of your experiments and their results and put them into a report. The report should include the following:

  • For each language and for each issue, sample code segments and the results of their execution. You should explain what your example does, clearly.
  • A section that includes your evaluation of these languages in terms of readability and writability of boolean expressions. Write a paragraph discussing, in your opinion, which language is the best in terms of boolean expressions. Explain why.
  • A section about your learning strategy. A learning strategy is an individual's approach to complete a task. In this section, discuss, in detail, the material and tools you used, experiments you performed. Also talk about personal communication, if you had. Give the URLs of the online compiler/interpreters you used to run your programs (10 points)
